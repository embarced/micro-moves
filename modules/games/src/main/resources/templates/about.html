About this Module

This module is implemented in Java with Spring Boot and uses Spring Web MVC, Spring Data and Thymeleaf. See Micro Moves blog series for details. It introduced this module in Part 1 as a plain REST service and enhanced it in Part 3 with this MVC UI.
