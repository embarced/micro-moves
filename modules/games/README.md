# games
Manages games between players, stores their moves.

## Technologies used
* Java 8
* Spring Boot 2.0 (Spring Web MVC, Spring Data ...)
* Thymeleaf

## Starting the Service standalone

./gradlew bootRun

(runs the Spring Boot WebApp on Port 8080)
