# chess-diagrams
Creates chess graphics, especially chess boards with positions

## Technologies used
* Python
* Pypthon-Libraries: Flask, Pillow
* Tests with pytest and tox

## Running the tests
* pip install tox
* tox

## Starting the service
* pip install -r requirements.txt
* python chess_diagrams.py
