# rules

Chess rules as a Web Service

## Technologies
* JavaScript
* Node.js  
* Unit Tests with Mocha

## Running the subsystem

### server
$ npm start

(static home page with example calls: http://localhost:8081/)

### Unit tests (Mocha)
$ npm test
