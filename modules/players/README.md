# players
Manages players and authenticates them.

## Technologies used
* Python
    * Pypthon-Libraries: Flask, Jinja2, TinyDB, PyJWT
* Tests with pytest and tox

## Running the tests
* pip install tox
* tox

## Starting the service standalone
* pip install -r requirements.txt
* python players.py
